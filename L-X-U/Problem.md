問題文
=====

Bi = U,D の時、Ai 以上の整数、Bi = D の時、Ai 以下の整数

- 2 <= N <= 10^5 × 2
- 1 <= Ai <= 10^9
- Bi = L,R のどちらか
- どのテストケースにもL,Rが必ず1度は登場する
