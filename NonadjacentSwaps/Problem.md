問題文
=====
英小文字のみからなる長さが N の文字列 S, T が与えられます．  

S に対し，次の操作を 0 回以上好きなだけ行うことができます：  
- S の互いに隣接しないような 2 文字の組を選び，それらを交換する  

S を T に一致させることはできますか？  
制約
-----
- 1 <= N <= 10^5

入力
-----
入力は以下の形式で標準入力から与えられる．
```md
S: abc, T: cba;  Yes
S: abcd, T: abcd;  Yes
S: tgc, T: agc;  No
S: bdcea, T: abcde;  Yes
S: a, T: z;  No
S: ab, T: ab;  Yes
S: ab, T: ba;  No
S: icpc, T: icfp;  No;
```

出力
-----
